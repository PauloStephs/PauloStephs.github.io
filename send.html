Tutorial de como mandar a poesia -

Mande uma mensagem para o endereço eletrônico poesiascms@gmail.com neste formato:

    Criação do perfil -

  • Nome de exibição
  • Gostos para poemas
  • (opcional) Anexar uma foto(quaisquer dimensões)

    • Mandar poemas -

  • Título da poesia
  • Texto

Se não for enviada uma foto de perfil, usaremos a padrão.

Caso você já tenha um perfil, fale na mensagem, para que não ocorra enganos.
